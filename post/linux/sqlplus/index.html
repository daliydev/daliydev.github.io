[linux] oracle 원격 접속 (sqlplus)

0001-01-01
linux

리눅스에서 오라클 접속 하기.

시작

  1. 두 개의 파일 다운로드
    1. ex) instantclient-basic-linux.x64-12.1.0.2.0.zip
    2. ex) instantclient-sqlplus-linux.x64-12.1.0.2.0.zip
  2. 서버 업로드하고 압축 해제
  3. 두 개의 파일이 한 곳에 해제됨 ex)instantclient_12_1
  4. 설치 받은 폴더 하위 디렉토리에 .bashrv 파일을 생성 후 실행
1
2

export LD_LIBRARY_PATH=/home/oracle/instantclient_12_1
source .bashrv

접속 방법

1

# ./sqlplus id/pw@ip:port/servicename

ORA-21561 : OID generation failed 문제 해결
/etc/hosts
127.0.0.1 localhost [호스트명]
호스트 명 : hostname

외부 스크립트 실행

1

# ./sqlplus id/pw@ip:port/sid < .sql
