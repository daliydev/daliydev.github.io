ls 현재 위치 목록 조회
-a 숨겨진 파일
-l 자세한 내용
-S 파일 크기 순으로 정렬
-r 거꾸로 출력
-R 하위 디렉토리까지 출력
-t 시간순으로 보기

cd 디렉토리 이동
cd ~ 홈디렉토리로 이동
cd .. 상위 디렉토리 이동
cd / 절대경로 이동
cd - 이동하기 바로전의 디렉토리 이동

mkdir 디렉토리 생성
mkdir -p depth1/depth2 존재하지 않는 하위디렉토리까지 생성

cp 파일 복사
cp [file1] [file2] file1 > file2 복사
cp -f [file1] [file2] 복사대상이 있으면 지우고 복사
cp -R [file1] [file2] 디렉토리안의 모든 하위경로까지 복사

mv 파일 이동
mv [file1] [file2] file1 의 파일을 file2 의 이름으로 이동/변경
-b file2 파일이 존재하면 백업한 뒤에 이동
-f 파일이 존재하면 덮어씀

rm 파일 제거
-f 묻지 않고 삭제
-r 디렉토리 삭제

cat 파일 내용 출력

touch 파일 날짜시간정보를 변경, 0인 빈 파일 생성
touch -c [file] 현재시간으로
touch -t YYYYMMDDhhmm [file] 날짜 변경
touch -r [file1] [file2] file2 파일을 file1 파일의 날짜 정보와 같게 변경

alias 현재 alias 출력
alias f='file' file 값의 새 명령어 f 생성
unalias [f] f라는 alias 해제
