[java] Interpreter 패턴

0001-01-01
java

인터프리터 패턴은 핸동 패턴 중 하나 이고, 일련의 규칙으로 정의된 언어를 해석하는 패턴이다.
예시로 자바 컴파일러나 정규표현식, 번역기 등이 있다.

구조
Context 모든 expression 사용되는 정보. 구문 분석
Expression 모든 노드에 해당하는 클래스들의 공통으로 가져야할 연산 메서드를 정의하기 위한 인터페이스

public interface Expression {
    int interpret(Map<Character, Integer> context);
}

public class Plus implements Expression {
    
    private Expression left, right;
    
    public Plus(Expression left, Expression right) {
        this.left = left;
        this.right = right;
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return left.interpret(context) + right.interpret(context);
    }
}

public class Minus implements Expression {
    private Expression left, right;
    
    public Minus(Expression left, Expression right) {
        this.left = left;
        this.right = right;
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return left.interpret(context) - right.interpret(context);
    }
}

public class Multiply implements Expression {
    private Expression left, right;
    
    public Multiply(Expression left, Expression right) {
        this.left = left;
        this.right = right;
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return left.interpret(context) * right.interpret(context);
    }
}

public class Division implements Expression {
    private Expression left, right;
    
    public Division(Expression left, Expression right) {
        this.left = left;
        this.right = right;
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return left.interpret(context) / right.interpret(context);
    }
}

public class Variable implements Expression {
    private Character character;
    
    public Variable(Character character) {
        this.character = character;
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return context.get(character);
    }
}

public class Context implements Expression {
    private Expression result;
    
    public Context(String expression) {
        Stack<Expression> stack = new Stack<>();
        
        for(char c : expression.toCharArray()){
            switch (c){
                case '+' : stack.push(new Plus(stack.pop(), stack.pop())); break;
                case '-' : stack.push(new Minus(stack.pop(), stack.pop())); break;
                case '*' : stack.push(new Multiply(stack.pop(), stack.pop())); break;
                case '/' : stack.push(new Division(stack.pop(), stack.pop())); break;
                default: stack.push(new Variable(c)); break;
            }
        }
        
        result = stack.pop();
    }
    
    @Override
    public int interpret(Map<Character, Integer> context) {
        return result.interpret(context);
    }
}

public class Main {
    public static void main(String[] args){
        String expression = "xyz+*";
        Context context = new Context(expression);
        int result = context.interpret(Map.of('x', 1, 'y', 2, 'z', 3));
        System.out.println(result); // 5
    }
}
  • Expression : 인터페이스. 서브 클래스에 공통으로 적용될 추상 Interpret() 메서드를 선언함.
  • Context : 문장려을 받아 처리하는 클래스.
  • Plus, Minus, Multiply, Division, Variable : Expression의 구현 클래스. expression은 하나의 기능만한다(SRP)
