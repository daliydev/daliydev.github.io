정렬 알고리즘

1. 버블 정렬(bubble sort)

  • 버블 정렬은 인접한 두 원소를 검사하여 정렬하는 알고리즘

public class Main {
    public static void main(String[] args){
        int[] arr = {5, 10, 1, 20, 2, 15, 4, 3};
        
        for(int i = 0; i < arr.length; i++){
            for(int j = 0; j < (arr.length - 1) - i; j++){
                if(arr[j] > arr[j + 1]){
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }
}

2. 선택 정렬(selection sort)

  • 앞쪽부터 정렬하는 방식. 최솟값을 찾아 선택하여 정렬하기 떄문에 비교 횟수는 많지만 교환 횟수는 적다.

public class Main {
    public static void main(String[] args){
        int[] arr = {5, 10, 1, 20, 2, 15, 4, 3};
        
        for(int i = 0; i < arr.length - 1; i++){
            int minimumIndex = i;
            for(int j = i + 1; j < arr.length; j++){
                if(arr[j] < arr[minimumIndex]){
                    minimumIndex = j;
                }
            }
            
            int temp = arr[i];
            arr[i] = arr[minimumIndex];
            arr[minimumIndex] = temp;
        }
    }
}

3. 삽입 정렬(insertion sort)

  • 앞에서부터 원소가 위치 할 곳을 탐색한다.

public class Main {
    public static void main(String[] args){
        int[] arr = {5, 10, 1, 20, 2, 15, 4, 3};
        
        for(int i = 0; i < arr.length; i++){
            int target = arr[i];
            int j = i;
            
            while(j > 0 && target < arr[j - 1]) {
                arr[j] = arr[j - 1];
                j--;
            }
            
            arr[j] = target;
        }
    }
}

4. 퀵 정렬(quick sort)

  • 피벗을 기준으로 두 개의 부분리스트. 큰 값/작은 값을 분리해서 정렬한다.
